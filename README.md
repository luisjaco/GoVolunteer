# GoVolunteer

GoVolunteer connects volunteers and organizations using a social media-like mobile application. 

**This is a group project for Intro to Software Engineering at New York Institute of Technology**

## Running

To run a local development server, simply
```bash
npm install     # install dependencies
npx expo start  # start the server
```
then scan the QR code in the [Expo Go app](https://expo.dev/go).
For more options, see the [Expo documentation](https://docs.expo.dev/more/expo-cli/#develop).
